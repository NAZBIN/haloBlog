# **《止损：如何克服贪婪和恐惧》感悟**

-   以为自己是在投资的，大多数是投机；以为自己是在投机的，大多数在赌博。
-   市场亏损应该是外部而客观的损失，一旦把个人情绪带进去，它才会变成主观损失。自我过度摄入，让你用一种负面的方式来看待，这个亏损就成为一项失败，是一个不好的事情，是一个错误

-   我要做的是投资：安全无虞的本金和适当的报酬
-   最至关重要的：入场前设定计划控制亏损、入场后客观的坚决执行，不引入主观情绪化的操作风格。

-   赌博：以金钱或其他财务进行跟运气有关的游戏
-   赌博活动是自行创造风险！而投资/投机则是承担以及管理本已存在的风险！
-   赌博、投资或交易并不是取决于活动本身，而是要看我们如何进行这些活动。
-   增兵，需要制定计划来执行决策。计划的有效与否，就看是否预先设定止损，以避免心理因素造成的损失。一旦不按照计划施行，就会触发潜在的心理因素，可能为亏损埋下伏笔。

-   进场前设定止损：要是在建仓后才设定止损点，或者开始调低止损位包容更多的亏损，或是改变决策时设定的基本条件，那么就是在：

-   投注或赌博，因为想证明自己是对的。
-   内化亏损
-   做出群众模式的交易，因为做出情绪化的决策，我的亏损会因此比原先愿意承担的还要严重得多。

-   警示录："相对于赚钱，我比较注意赔钱。别光想赚钱，要专注于保护你的战果。"

-   只有交易才会有这种奇怪现象，让一些没有真才实学毫无优势的人也可以获得短暂成功，因此让人误以为运气比技巧还重要。
-   事实上交易成不成功，通常是因为心理因素，而不是战术问题。
-   屡次的失败，只要不被打垮，就是成功的基础；反之，要是接二连三的胜利就志得意满地情绪化看待，反而是失败的开始。
-   一连串的成功之后造成心理扭曲。会把交易仓位跟'自负'绑在一起，而陷于惨重的亏损
-   "失败最简单的方法，就是开始确信自己很成功"
-   巴菲特："你的投资要集中。要是你有40个小老婆，那么你对谁都不会深入了解"
-   "对于所投资的企业，一定要了解它的业务，不然一旦股价下跌，就难以判断是否应该抄底。
-   "不要在底部钓鱼"
-   "甭想买到最低点、卖到最高点"
-   "大多数菜鸟散户（群众）在亏损还算少的时候死死抱着不放，他们原本可以在受伤不重的情况下退出，却受制于情绪和人性，造成亏损越来越大，终致损失惨重"

-   坚持自己交易系统的基本架构，上层不断改进完善："世界上有那么多赚钱的方法，你可能一辈子都在尝试新方法，然后屡战屡败，因为你不知道怎样才能不赔钱！"

-   选股：了解主力操盘风格，  其重点是看主力的拉升风格。

-   有80%的低吸成功率后再建头仓入场。
-   用头仓去试探是否开启主升浪。没有非常大的上涨行情是绝对不能全仓介入。

-   思考：

-   "尽早砍掉亏损"

-   到底是什么意思？
-   只要亏损就要把仓位处理掉吗？
-   任何投资或交易仓位在某些时候都可能出现亏损，又怎么知道这是个必须尽快处理掉的"亏损"，因为价位不会再回来，仓位不可能赚钱？

-   "不要跟随大众，要逆向而行。"

-   大众意见以谁为准？
-   如何判断大众动向？
-   况且，跟大家唱反调也未必见效，有时候于群众背道而驰也会惨遭淘汰出局。
-   情绪本身是无法避免的，但情绪化可以也应该避免。情绪化是根据情感起伏来做决策，而最容易做情绪化决策的，就是群众。

-   那为什么市场上的人通常很难主动认赔呢？

-   这是因为损失总是被看做 "失败" ，在生活中其他领域里一样， "损失"这个词都带有负面含义。我们往往把损失等同于错误、不好喝失败，而将胜利视为正确、善良和成功。我们在市场上亏了钱，也会认为自己做错了才会赔钱。
-   参加比赛输了，未必是因为做错什么，也许只是被击败了。旁观者（下注人）输了，那么对于这场比赛你也许是赔了钱（预测错误），但未被击败。

-   外在损失与内在损失：

-   内在损失：自我控制、尊严、爱、理智
-   外在损失：赌注、游戏或比赛、金钱
-   外在损失与内在损失的关系：

-   外在损失是客观的，内在损失是主观的。
-   外在损失是个客观事实，内在损失是根据个人经验来界定。
-   客观事实不会因为你的感受或反应而改变。这个事实只能被接受。

-   负面含义也可以正面看待：

-   亏损不见得是错误，就算是一个很好的决策！也可能产生亏损！
-   在交易市场上，亏损应当被看做是坏灯泡或烂苹果，只是业务的一部分，必须淡定处之。

-   市场亏损应该是外在损失，只有把个人情绪带进去，它才会变成主观的内在损失。

-   市场亏损是如何转化成内在损失的：

-   了解其转换的关键，是先分清"事实"与"意见"的细微差别。

-   事实是已获客观证实的事物，事实没有对错。
-   意见是一种个人评估和盘算，意见的对错则取决于是否符合事实，所以有对错之分的是意见。

-   语意上的混淆，很容易把市场上赔钱看成是一个错误，如此，就把和金钱有关的决策（外在）看作是自己的名誉和尊严（内在）
-   这就是自我意识的介入，开始以个人情绪看待市场。
-   于是亏损也从客观存在变成主观认定，这就不是损失一笔金钱而已！而是一种个人的内在损失（自我控制、尊严、理智）

-   内在损失的五个阶段：

-   否定："这怎么会下跌呢？应该拉起的啊"
-   愤怒："隔天怎么还在跌！！"
-   讨价还价："只要能涨回去，哪怕涨回去一半我就清仓！！"
-   抑郁："睡不着吃不下，无法专注于工作，更不愿听从那些叫我赶快认赔出场的意见。"
-   接受

-   生命中没有什么事情是保证成功，或者一定会照你的意思来走。感兴趣的风险，当然是在金融方面，会产生金钱亏损的风险。

-   并非所有参与赌博游戏的人都是赌客。所谓的"职业赌徒"，也是以赚钱为目的，在押注时细心、专注且严守纪律，跟一般赌客完全不同。他们的押注有系统化的方法，通常只有在罕见且非常有利的状况下才会出手。
-   职业赌徒严格的控制自己的行为，对于他所选择的赌戏通常研究的非常透彻，才会有此表现。
-   要是有人热衷于追求创业冒险，或者是在金融市场上寻求刺激，那么不管他对结果有多大的控制程度，都是在赌博。
-   如果你追随每个相关讯息，盯着屏幕上的价位跳动，就像烛火般随风摇拽飘忽，你就是市场群众之中的一员。

-   个人与群众的基本区别：

-   个人：经过理性思考、深思熟虑和分析之后才采取行动。
-   群众：根据感觉、情绪和冲动来行动。
-   个人会仔细考虑自己的意见，群众则是受到情绪化观点所影响，而非理性思考。
-   个人在群众之中，摄取了群众的能量，因为自己隐身其中（或说融为一体），就会做出一些平常不会做的事情。身处群众之中，让个人控制自我的责任感消失无踪（球赛胜利后，群众冲进球场，连球门都会推倒）。

-   杂记📝：
-   盯着屏幕最底下的股价跑马灯，或者实际在市场上感受行情的上下起伏，这都像是注视催眠师那条来回摆动的表链。
-   群众特征中最不可理解者之一是，尽管面对如山铁证的挑战，成员们仍然可以固执地坚守错误假设。所以，明明看到亏损一直扩大，还是死抱着市场仓位不放，这就是群众。
-   要是你对市场仓位的处置跟你原本预想的不同，那你就是心理意义上的群众中的一员，不管你自己有没有意识到，你所做的交易其实是一种群体交易。不然的话，你早就根据自己的判断作出该做的事。
-   那些想很快补回亏损的人往往会赔的更多，是因为他处于怀着期待的关注下，一听到谁报出名牌就急着杀进去。极端情绪化，注定要亏损。
-   声望是一种个人、作品、意念或愿望对我们施以主宰的方式。它能完全瘫痪我们的辨识功能，让我们心中、眼中只能看得到、感受的到惊叹。
-   一旦市场行情与你所料，你看起来就是个英雄，你是如此厉害（声望），大家对你于阿谀奉承。情感和情绪淹没了你（感染），你就这么被催眠了。
-   情绪本身没有什么好或坏，情绪只是情绪而已。我们要避免的是情绪化，而非情绪。

-   为了证实正确而参与市场，那叫做投注，而追求兴奋刺激则是赌博，如果是要投机，那么按照定义，你得先有个计划。
-   股市每日只有涨跌两种结果。涨则盈利跌则亏损，亏损意味着损失并不意味着失败，一个好的执行策略来降低大的损失，不造成内在损失即为PLAN的首要原则。
-   赚钱仓位可能因此迟迟不放手，结果从赚钱拖成赔钱，而亏损仓位则是一拖再拖，一延再延，使得亏损远远超出原本设定的额度。
-   而大众几乎都是在进入市场之后才选择止损或出场机制，有些人根本没有设定出场条件。
-   市场上赔钱的"理由"（更精确地说，就是"借口"）有千百种，而欠缺计划或不按计划施行，往往就是根源所在。
-   要清醒的意识到：即使按照计划施行也会有亏损的情况，这是再正常不过的，但不按计划施行是加重亏损的根源所在。
-   "不管怎么看，你进入市场的时机都不会绝对完美，因此某些时候总要呈现失利的情况，你就有许多机会限于情绪化的决策，进而导致亏损。"
-   在决策时保持客观。
-   不管要做什么决策，若想提升决策效率，就要通过计划以保持客观。
